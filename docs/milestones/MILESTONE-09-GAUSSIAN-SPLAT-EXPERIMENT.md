# 目标 9：高斯泼溅独立实验

完成日期：2026-07-16

## 实验边界

- 独立入口：`/examples/experiments/gaussian-splat-experiment.html`。
- 不接入导演台 store、对象树、模型导入器、工程 JSON 或二创协议。
- 主导演台不会加载 Spark；只有进入实验页时才加载高斯渲染代码。
- 使用 MIT 协议的 `@sparkjsdev/spark 2.1.0`，与当前 Three.js 版本兼容。
- 测试场景由项目代码确定性生成，不使用来源不明的第三方扫描资产。

## 实验功能

- 生成 16,000 个 Gaussian Splat，组成环体、中心球体和地面网格。
- 显示 splat 数量、原始体积、加载耗时、实时 FPS 和坐标约定。
- 使用 Three.js `PerspectiveCamera` 和 `OrbitControls`，坐标约定为 Y 轴向上。
- 支持重置视角、自动旋转开关和 PNG 静帧导出。
- 运行报告记录相机位置、目标、FOV、GPU、画布尺寸、加载耗时和 FPS，供浏览器验证读取。

## 浏览器实测

设备：Apple M5 Pro，Chrome 150，1280×720 CSS 视口。

- Splat 数量：16,000。
- 原始 `.splat` 数据：512,000 字节（500 KB，每点 32 字节）。
- 加载耗时：约 50–65 ms。
- 实时帧率：约 57–70 FPS。
- 实际渲染画布：1920×1080（DPR 1.5）。
- 相机：位置约 `[4.8, 2.8, 6.4]`，目标 `[0, 0.45, 0]`，FOV 52°。
- 控制台业务错误：0。

画布像素检查：

- 截图尺寸：1280×720。
- 亮像素占比约 31.6%。
- 彩色像素占比约 15.0%。
- 结果确认不是黑屏或空画布。

## 包体结论

多入口生产构建结果：

- 主导演台入口 JS：约 0.98 MB。
- 共享 Three.js chunk：约 0.73 MB。
- 高斯实验入口 JS：约 4.95 MB，gzip 约 1.76 MB。

Spark 功能完整，支持 PLY、SPZ、SPLAT、KSPLAT 和 SOG，但实验包体明显偏重。当前结论是继续保持独立实验，不加入主编辑器首屏依赖。
