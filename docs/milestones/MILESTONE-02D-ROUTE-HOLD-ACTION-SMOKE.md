# 目标 2D：人物路线停留动作真实验收

完成日期：2026-07-17

## 修复内容

- 新增共享 `getObjectMotionActionSample`，同时返回当前动作、动作播放秒数和停留点索引。
- “指定停留动作”从人物到达停留点时的 `0 秒` 起播，不再按整条时间轴从任意中间帧开始。
- Mixamo FBX、UE4 人偶、程序人偶共用同一动作播放时间规则。
- “保持当前动作”继续沿用原路线动作时间轴；“站立”保持无动作；普通移动段节奏不变。

## 独立验收

- 入口：`../../examples/experiments/route-action-smoke.html`。
- 真实人物：Camille FBX。
- 路线：移动到中点 -> 停留挥手 2 秒 -> 恢复走路并继续移动。
- 视图：导演主视口、成片监看小窗、第一视角、看成片、视频导出。
- 阶段：移动前段、刚到停留点、停留中段、离开后移动、重新播放停留中段。

## 浏览器结果

- 状态：`passed`。
- 到达中点：约 `4.0 秒`；离开中点：约 `6.0 秒`；停留准确为 `2 秒`。
- 到点动作：`wave-cycle`，动作播放时间 `0 秒`。
- 停留中段位置 X=`0`，挥手时间 `1 秒`，右手姿势相对移动段变化 `0.619295`。
- 离开后恢复 `walk-cycle`，位置 X=`0.599998`，证明继续前进。
- 五视图骨骼与人物位置最大差：全部 `0`。
- 重播停留中段姿势差、位置差：全部 `0`。
- 浏览器业务错误：`0`。
