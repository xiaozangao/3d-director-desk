# 目标 6A：全景图恢复与多视图一致性

完成日期：2026-07-17

## 已完成

- 在“3D场景 > 背景”增加全景图导入入口，支持 JPG、PNG、WEBP。
- 非 2:1 图片会经过现有适配流程；大图片二进制保存到 IndexedDB，项目状态仅保存稳定引用。
- 支持更换、删除、左右旋转、亮度调整和一键恢复默认方向。
- 主导演视图、第一视角、成片监看小窗和参考视频导出统一使用当前全景资产。
- 恢复宿主 `storyai:director-desk-panorama` 消息接入，保留来源校验和导演台实例隔离。
- 删除或更换项目中的全景引用时保留本地二进制，使 Ctrl+Z 和历史场景仍可恢复原图。
