# 目标 7D：Windows 基准工具交付收口

完成日期：2026-07-17

## 用户入口

- 可见性能档位严格收敛为三项：自动、流畅、高清。
- 旧版本持久化的“均衡”会迁移为自动，不会出现没有选中项的状态。
- 均衡仍作为自动档内部状态和固定基准参数保留，保证旧报告和重复测试可比较。
- 基准页增加常驻状态条，不需要重新打开性能面板即可看到预热、采样、完成、平均 FPS、1% Low 和下载按钮。
- 轻量、中等、重型测试继续在干净 URL 和临时工作区运行，不携带导演台 ID、令牌或当前工程内容。

## 匿名报告 v3

- 公开 schema 从 v2 升级为 v3。
- 新增必要的非隐私上下文：测试场景、性能档和软件版本。
- 继续使用逐字段白名单，只包含 FPS、帧时间、浏览器大版本、标准化系统类型、GPU、逻辑线程、固定场景规模和画布尺寸。
- 不包含用户名、时间戳、工程名、导演台 ID、模型名、URL、Cookie、IP、本地路径、完整 UA、原始平台字符串或工程 JSON。
