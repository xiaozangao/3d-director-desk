# 目标 6B：地面材质、编辑网格与路线防穿模

完成日期：2026-07-16

## 已完成

- 新增五种地面材质预设：摄影棚、混凝土、柏油、木地板、草地。
- 地面使用实时生成的可重复 Canvas 纹理，不依赖外部图片；支持粗糙度和金属度差异。
- 切换材质时自动释放旧纹理，关闭地面时不创建无用纹理。
- 保留地面颜色、亮度、透明度和高度的手动微调。
- 将以下状态拆成互不影响的独立设置：
  - 显示编辑网格。
  - 移动时吸附网格。
  - 显示地面。
  - 启用地面和场景碰撞（界面简称“路线防穿模”）。
- 编辑网格只显示在主交互视图，不进入成片监看或参考视频导出。
- 路线防穿模继续同时约束人物移动路线和镜头路线，并可独立于地面显示开关启用。
- 旧项目自动迁移到默认摄影棚材质并保持网格可见。
